# FirstCommit

**Learn technology by working with it.**

FirstCommit is a free, open-source educational project dedicated to helping people become confident builders, problem-solvers, and lifelong learners through practical, project-based learning.

Our goal isn't simply to teach programming more efficiently. it's to help people understand the technology they use every day and gain the confidence to solve problems with it.

## Why FirstCommit?

Hi! My name is Joshua, and like many beginners, I spent years searching for the "perfect" way to learn programming. I tried Khan Academy, Codecademy, Codédex, freeCodeCamp, Boot.dev, and many other resources. Every platform had something valuable to offer.

Yet I kept running into the same problems:

* Many courses spent too much time on toy examples before helping you build something meaningful.
* The number of tutorials, bootcamps, and learning paths made it difficult to know where to start.
* Even after completing lessons, I often understood the syntax better than I understood how to actually use the technology to solve problems.

The real goal is becoming someone who can look at a problem and think:

*"Technology can help me solve this problem more efficiently and better than I could by myself."*

That might mean:

* Building a website from scratch.
* Writing a Python script to automate hundreds of repetitive tasks.
* Using browser developer tools to understand how a webpage works.
* Navigating efficiently with the command line.
* Deploying an application.
* Learning a new language or framework when a project requires it.

FirstCommit exists to make that journey to building real meaningful projects easier.

## Our Philosophy

We believe people learn best when they:

* Build projects that solve real problems.
* Learn concepts when they naturally become useful.
* Focus on understanding instead of memorization.
* Develop mental models instead of copying code.
* Learn by experimenting, building, and teaching others.
* Read explanations written by people who still remember what it felt like to be beginners.

## Contributing

FirstCommit was started by someone who is still learning—and that's intentional.

Many educational resources are written by experts who have forgotten what was confusing when they first encountered a concept. We want to preserve those moments where an idea finally **clicks** while they're still fresh.

If you've recently learned something that made you think,

*"I wish someone had explained it like this sooner."*

then your perspective is valuable.

Whether you're fixing a typo, improving an explanation, writing a guide, contributing code, or sharing a project, every contribution helps make technology more approachable for the next learner.
